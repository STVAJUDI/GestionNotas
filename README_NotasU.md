# NotasU – Aplicativo académico de notas (MVP académico)

## Idea
Aplicación web para que estudiantes consulten sus **materias** y **notas por semestre**, con promedio del periodo y reporte PDF (simulado). Incluye rol docente para carga de notas (simulada).
